## Python - Clases and Objects 

This repository contains Python programs that demostrate Object-Oriented Programming (OOP) principles through the implemetantion of a "Square" class. Each task progressively builds upon the previousone to enhance functionality and enforce best parctices in encaptulation, validation, and methos implemetation.\

## Task Explanation

0) My first square

1) Square with size

2) Size Validation

3) Area of a square

4) Access and update private attribute

5) Printing a square

6) Coordinates of a square
